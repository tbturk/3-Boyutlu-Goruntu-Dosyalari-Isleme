			3 BOYUTLU GÖRÜNTÜ DOSYALARI PROJESİ
PROJE TANIMI:
---------------------------------------------------------------------------------
	Bu uygulamada farklı türdeki 3 boyutlu görüntü dosyalarından kontrollü 
okuma yapılarak, dosyanın içindeki veriler kullanılıp son kullanıcının istediği
problemler çözülmüştür.
---------------------------------------------------------------------------------

GELİŞTİRME ORTAMI:
---------------------------------------------------------------------------------
	Proje Windows platformunda C dili kullanılarak yazılmış olup; Code:Blocks
ortamında, gcc kullanılarak derlenip test edilmiştir.	
---------------------------------------------------------------------------------

KULLANIM NOTLARI:
---------------------------------------------------------------------------------
0-	Test edilecek dosyaları main.c nin bulunduğu klasöre kopyalayın(projenin
klasörü)
1-	Projeyi Code:Blocks üzerinde Relase Modda çalıştırın.
2-	Giriş arayüzü, projenin içinde olduğu klasördeki ".nkt"
uzantılı dosyaları ekranda listeler.
3-	Karşınıza gelen giriş arayüzünde işlem yapmak istediğiniz ".nkt" uzantılı
dosyanın satır numarasını girdi olarak girin.
(ÖR:	Satır: 5-ornek.nkt		Girdi:	5+enter)
4-	İşlem seçim ekranı, seçilen dosyanın üzerinde yapılabilecek işlemleri
listeler.
5-	Karşınıza gelen işlem seçim ekranında yapmak istediğiniz işlemin satır
numarasını girdi olarak girin. "Dosya Kontrolü" işlemi dosyadaki hataları tara-
dığından dolayı bu işlemi yapmadan diğer işlemlere geçemezsiniz.
(ÖR:	Satır: 1.)Dosya Kontrolü		Girdi:	1+enter)
6-	Eğer tüm işlemleri yaptıysanız 6'yı girdi olarak girip sıradaki nkt dos-
yasını seçebileceğiniz giriş arayüzüne tekrar dönebilirsiniz.
7-	Tüm nkt dosyalarıyla işlemleriniz bittikten sonra giriş arayüzüne dönüp
0'ı girdi olarak girerseniz programı sonlandırabilirsiniz.
8-	Programdan çıkmak için giriş arayüzünde 0'ı girdi olarak girmelisiniz.
---------------------------------------------------------------------------------
